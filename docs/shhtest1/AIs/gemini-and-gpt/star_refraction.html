T762171 星图折射

题目背景

观星台的镜面会把星光折射到另一张坐标纸上。原本呈菱形扩散的星光，在折射图上会变成规整的方窗。

现在给出许多带能量的星点，以及若干次观测。每次观测会选定一个中心和一个曼哈顿半径，你需要求出落在观测范围内的星光总能量。

题目描述

平面上有 \(n\) 颗星，第 \(i\) 颗星坐标为 \((x_i,y_i)\)，能量为 \(w_i\)。

有 \(q\) 次询问。每次给出中心 \((a,b)\) 和半径 \(d\)，你需要输出所有满足

\[ |x_i-a|+|y_i-b|\le d \]

的星点能量之和。

输入格式

第一行两个整数 \(n,q\)。

接下来 \(n\) 行，每行三个整数 \(x_i,y_i,w_i\)。

接下来 \(q\) 行，每行三个整数 \(a,b,d\)。

输出格式

对每个询问，输出一行一个整数，表示能量之和。

输入输出样例 #1

输入 #1

5 3
0 0 5
1 1 3
2 0 4
-1 2 7
3 3 6
0 0 1
1 1 2
2 2 3

输出 #1

5
12
20

说明/提示

样例解释

第三次询问中心为 \((2,2)\)，半径为 \(3\)。满足条件的星点为 \((1,1),(2,0),(-1,2),(3,3)\)，能量和为 \(3+4+7+6=20\)。

数据范围

对于所有测试点：

  • \(1 \le n,q \le 2\times 10^5\)
  • \(-10^9 \le x_i,y_i,a,b \le 10^9\)
  • \(1 \le w_i \le 10^9\)
  • \(0 \le d \le 2\times 10^9\)

计分方式

本题共 20 个测试点，每个测试点 5 分。

时间与空间限制

  • 时间限制：1000 ms
  • 空间限制：128 MB
