T752957 求解二元一次方程

题目描述

输入一个二元一次方程的一般形式的二次项系数\(a\)、一次项系数\(b\)、常数项\(c\)，则相当于输入一个方程：\(ax^2+bx+c=0\)

你需要求出x的2个解。注意：两个解可能相等，当无解时，输出：“NoAns”。

输入格式

一行三个数，\(a,b,c\)，含义见题目描述。

输出格式

有解时，一行2个数，为方程解。保留2位小数。

无解时，输出一个字符串“NoAns”。

输入输出样例 #1

输入 #1

输出 #1

说明/提示

数据范围

子任务编号 范围 依赖
1 \(-10^5<a,c<10^{5},b=0\) 无
2 \(-10^5<a,b,c<10^{5},b=0\) 任务1

:::info
如果你没有学到解一元二次方程,这里是公式：

判根公式：\(\Delta=b^2-4ac\)

当\(\Delta<0\)时，无实根。

当\(\Delta==0\)时，两个根相等。

当\(\Delta>0\)时，两个根不相等。

若有根，求根公式：\(x=\frac{-b\pm\sqrt{\Delta}}{4a}\)

:::
