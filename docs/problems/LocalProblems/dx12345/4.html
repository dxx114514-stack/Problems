T659675 BrainF**k

题目背景

有一天小明在网上冲浪时看到了一款名叫BrainF**k的编程语言,但是苦于没有编译器，所以说他无法亲身体验。现在请你写一个解释器，用来解释小明输入的代码。

题目描述

BrainF**k会先给你一个长度为3万数组，范围是0~255。还会给你一个指针，用于指向你现在正在操作的元素，初始为最开始的元素。数组中所有数据初始化为零。

他的语法有：用<向左移动指针，用>向右移动指针。用+给现在指针所指向的元素的数值加一，用-给现在指针所指向的元素减一。使用[]循环，循环直到当前指针所指向的元素等于零，使用,来从用户输入一个字符（保证一定有输入），使用.来输出一个字符到控制台（强转）。没有在上面提到的字符则是全部为注释。为方便理解，设代码中包含n个,和m个. ，代码长度为S，此处输入输出的皆为数。另外，你需要处理指针越界的情况。又因你写的是解释器，所以在那之前要处理代码。越界时输出“SF”并终止。当一个循环超过255次时，认定其为死循环，输出“TLE”并终止。（具体参见：https://b23.tv/kR5bCwM）

输入格式

输入第一行，一个不包含空格的字符串，就是这个语言的代码。

接下来一行，n个数作为输入。

输出格式

一行,m个数,代码的输出。每个输出间空格分隔

输入输出样例 #1

输入 #1

<>,++.
1

输出 #1

SF

输入输出样例 #2

输入 #2

>,++.
1

输出 #2

3

输入输出样例 #3

输入 #3

>>,<<,++>>[-].<<.
1 2

输出 #3

0 4

输入输出样例 #4

输入 #4

>>++.<<<

输出 #4

2 SF

输入输出样例 #5

输入 #5

+[>+]

输出 #5

TLE
