T667417 简单计算题

题目描述

给定数组\(A\)，
计算\(\sum(A_x+A_y)^2\)，\(1 \le x < y \le n\)。

输入格式

本题有多组数据。

第一行一个整数\(T\),表示数据数量。

对于每组数据，输入两行：

·第一行一个整数\(n\)，表示数组\(A\)的长度\
·第二行\(n\)个整数，第\(i\)个数表示\(A_i\)。

输出格式

对于每组数据，输出一个整数，表示\(\sum(A_x+A_y)^2\)的值。

输入输出样例 #1

输入 #1

2
2
3 4
3
5 8 2

输出 #1

49
318

说明/提示

对于\(20\%\)的数据保证\(1 \le T \le 50,1 \le n \le 100,1 \le A_i \le 10^3\)。

对于\(50\%\)的数据保证\(1 \le T \le 100,1 \le n \le 600,1 \le A_i \le 10^4\)。

对于\(100\%\)的数据保证\(1 \le T \le 500,1 \le n \le 10^5,1 \le A_i \le 10^4\)。

注：本题输入输出量较大，请使用较快的读入方式。
