T627781 数位交换

题目描述

小明有两个\(N\)为正整数\(A=\overline{A_1A_2......A_{N-1}A_{N}}\)\
以及另一个\(N\)位正整数\(B=\overline{B_1B_2......B_{N-1}B_{N}}\)\
对于\(1 \le i,j \le N\)小明可以任意交换\(A_i\)和\(B_j\)。\
若干次操作后，\(A\)会变为\(A'\)，\(B\)会变为\(B'\)\
小明想知道他交换若干次后，\(A' \times B'\)最大可能是多少。\
小明认为这道题太烦了，于是想让精通\(c++\)的你帮他解决这个问题。

输入格式

第一行一个整数\(N\)\
第二行一个十进制\(N\)位整数\(A\)\
第三行一个十进制\(N\)位整数\(B\)\
题目保证没有前导零

输出格式

一行一个整数，表示\(A' \times B'\)的最大值，小明知道答案很大，于是请你对\(100000007\)取模。

输入输出样例 #1

输入 #1

3
123
456

输出 #1

342002

说明/提示

样例解释

过程如下：
|交换次数|\(i\)|\(A_i\)|\(j\)|\(A_j\)|\(A'\)|\(B'\)|\(A' \times B'\)|
|:-:|:-:|:-:|:-:|:-:|:-:|:-:|:-:|
|\(1\)|\(1\)|\(1\)|\(3\)|\(6\)|\(623\)|\(451\)|\(280973\)|
|\(2\)|\(3\)|\(3\)|\(2\)|\(5\)|\(625\)|\(431\)|\(269375\)|
|\(3\)|\(3\)|\(5\)|\(3\)|\(1\)|\(621\)|\(435\)|\(270135\)|
|\(4\)|\(2\)|\(2\)|\(3\)|\(5\)|\(651\)|\(432\)|\(281232\)|
|\(5\)|\(2\)|\(5\)|\(1\)|\(4\)|\(641\)|\(532\)|\(341012\)|
|\(6\)|\(2\)|\(4\)|\(2\)|\(3\)|\(631\)|\(542\)|\(342002\)|

数据范围

对于\(12\%\)的数据保证\(1 \le N \le 5\)\
对于\(36\%\)的数据保证\(1 \le N \le 500\)\
对于\(100\%\)的数据保证\(1 \le N \le 5 \times 10^5\)
