T609542 单词接龙长度

题目描述

\(Asher\)和他的朋友一起玩单词接龙游戏，规则如下：\
\(1.\)接龙队列中第\(i\)个单词的首字母要和第\(i-1\)个单词的结尾字母相同才可以。\
\(2.\)规定单词的编号是输入时的顺序。即单词\(s_m\)的编号是\(m\)。接龙队列里对于任意\(i,j\)满足\(1\le i < j \le k\)（\(k\)表示接龙队列的长度）都有\(s_i\)的编号小于\(s_j\)的编号。\
\
但是由于\(Asher\)和朋友们英语不好，他们只会\(n\)个单词，假设每个人都会这\(n\)个单词。\
现在给定\(n\)，以及\(Asher\)和他的朋友们都会的单词。\(Asher\)想知道上述规则中满足条件的\(i\)最大是多少？或者说，这个接龙队列最多有多少项？

输入格式

第一行，一个整数\(n\)，表示\(Asher\)和朋友们会的单词数量。\
第二行，\(n\)个字符串\(S_k\)，表示\(Asher\)和他的朋友会的单词。

输出格式

一行，一个整数，表示答案。

输入输出样例 #1

输入 #1

5
cat car dog tac coat

输出 #1

3

说明/提示

样例解释：

单词接龙队列如下：\
\(cat->tac->coat\)

数据范围：

本题采用\(subtask\)捆绑测试，你需要完成相应子任务的所有测试点，才能拿到该测试点相应的分数
|子任务编号 |分值 |数据范围 |
| :----------- | :----------- | :----------- |
|子任务\(1\) |\(20\)分 |\(1\le n \le 10\) |
|子任务\(2\) |\(20\)分 |\(1\le n \le 10^3\) |
|子任务\(3\) |\(60\)分 |\(1\le n \le 5×10^5\) |

保证每一个单词长度不超过\(15\)，每个单词都只由小写字母构成
