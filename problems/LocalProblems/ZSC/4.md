# U671454 ZSC的分块1

## 题目背景

$ZSC$天天无聊

## 题目描述

$cnt(i,f(i))=\sum_{k=1}^{n}[f(i)=f(k)]$ 

计算$\sum^{n}_{i=1}(| cnt(i,\lceil \frac{n}{i}\rceil)-cnt(i,\lfloor \frac{n}{i}\rfloor)|)\lceil \frac{n}{i}\rceil\lfloor \frac{n}{i}\rfloor \; mod \; 2^{64}$

## 输入格式

输入一个数，表示$n$

## 输出格式

输出一个数，表示结果

## 输入输出样例 #1

### 输入 #1

```
54188
```

### 输出 #1

```
2407821090
```

## 输入输出样例 #2

### 输入 #2

```
1
```

### 输出 #2

```
0
```

## 说明/提示

$n\leq 10^{14}+9$

对于$n\leq1000000 $你只有$90ms$
