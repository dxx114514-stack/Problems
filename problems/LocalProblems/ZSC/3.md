# T633823 A not too easy problem

## 题目背景

$ZSC$要为难入门的~~小朋友~~

## 题目描述

请你求$m^n$ $mod$ $10^{9}+7$的结果

## 输入格式

输入两个整数$m,n$意义如上

## 输出格式

输出一个数$m^n$ $mod$ $10^{9}+7$的结果。

## 输入输出样例 #1

### 输入 #1

```
1 1
```

### 输出 #1

```
1
```

## 输入输出样例 #2

### 输入 #2

```
20 0
```

### 输出 #2

```
1
```

## 输入输出样例 #3

### 输入 #3

```
10 8
```

### 输出 #3

```
100000000
```

## 说明/提示

本题采用子任务方式评分！\
对于$10\%$的数据保证$0 \le n \le 10^{9}$，保证$n=0$或$m=0$\
对于$30\%$的数据保证$0 \le n \le 10^{18}$\
对于$50\%$的数据保证$0 \le n \le 10^{38}$\
对于$100\%$的数据保证$0 \le n \le 10^{2000},0 \le m \le 20$
