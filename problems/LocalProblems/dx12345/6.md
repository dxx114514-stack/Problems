# T752958 求长方形每种块数量

## 题目描述

给出长方体的长$a$，宽$b$，高$c$，对于其每个棱，作它的平行线，每2个平行线中隔1个单位距离。

现在为这个长方体外表面涂上色，后将长方体沿线切割开来，使其分成很多正方体。

你需要求出3面涂色、2面涂色、1面涂色和不涂色的正方体数量。

## 输入格式

一行三个数，$a$,$b$,$c$,含义见题目描述。

## 输出格式

分别输出3面涂色、2面涂色、1面涂色和不涂色的正方体数量。

## 输入输出样例 #1

### 输入 #1

```
4 3 2
```

### 输出 #1

```
4 9 4 0
```

## 说明/提示

### 数据范围

对于100%的数据，$2 \leq a,b,c \leq 10^{18} $
