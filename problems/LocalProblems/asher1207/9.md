# T604672 四元组

## 题目描述

给定一个数$N$，已知$a,b,c,d$均为正整数，求满足$ab+cd=N$的四元组$\{a,b,c,d\}$的数量

## 输入格式

一行，一个正整数$N$

## 输出格式

求满足条件的四元组$\{a,b,c,d\}$的数量

## 输入输出样例 #1

### 输入 #1

```
4
```

### 输出 #1

```
8
```

## 输入输出样例 #2

### 输入 #2

```
475
```

### 输出 #2

```
15624
```

## 说明/提示

## 样例解释：
对于第一组样例，有：\
$\{1,1,1,3\}$\
$\{1,1,3,1\}$\
$\{1,2,1,2\}$\
$\{1,2,2,1\}$\
$\{1,3,1,1\}$\
$\{2,1,1,2\}$\
$\{2,1,2,1\}$\
$\{3,1,1,1\}$
