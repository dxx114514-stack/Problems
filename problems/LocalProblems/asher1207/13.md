# T619371 B2107 图像旋转水题（速刷）

## 题目描述

输入一个$m$行$n$列的黑白图像，将它顺时针旋转$180°$后输出。

## 输入格式

第一行包含两个整数$n$和$m$，表示图像包含像素点的列数和行数。$1 \leq n \leq 100,1 \le m \le 100$。

接下来$m$行，每行$n$个整数，表示图像的每个像素点灰度。相邻两个整数之间用单个空格隔开，每个元素均在$0∼255$之间。

## 输出格式

$m$行，每行$n$个整数，为顺时针旋转$180$度后的图像。相邻两个之间用单个空格隔开。

## 输入输出样例 #1

### 输入 #1

```
3 3
1 2 3
4 5 6
7 8 9
```

### 输出 #1

```
9 8 7
6 5 4
3 2 1
```
