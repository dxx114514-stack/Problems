# T619712 最长公共子序列

## 题目描述

给定$n$和$1～n$的两个排列，求这两个排列的最长公共子序列。

## 输入格式

第一行一个正整数$n$\
接下来两行，各$n$个整数，表示$1～n$的两个排列

## 输出格式

一行，一个正整数，表示两个排列的最长公共子序列

## 输入输出样例 #1

### 输入 #1

```
5
2 4 5 3 1
5 2 1 4 3
```

### 输出 #1

```
3
```

## 说明/提示

### 样例解释
最长公共子序列是$\{2,4,3\}$
