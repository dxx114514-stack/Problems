# T613900 末尾好数

## 题目描述

假设$A,2A,3A,4A……98A,99A$的末两位互不相同，则称$A$为“末尾好数”。求$1$到$n$的“末尾好数”个数。

## 输入格式

本题有多组测试数据。\
第一行一个正整数$t$，表示测试数据个数。\
对于每组测试数据，一行一个正整数$n$，含义见题目描述。

## 输出格式

对于每组数据，一行一个整数，表示答案。

## 输入输出样例 #1

### 输入 #1

```
3
1
8
19
```

### 输出 #1

```
1
3
8
```

## 说明/提示

对于$24\%$的数据，保证$1\le n \le 10^4$,$1 \le t \le 40$\
对于$100\%$的数据，保证$1 \le n \le 9×10^{18}$，$1 \le t \le 10^5$
