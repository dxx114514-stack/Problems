# T604413 整数裂项

## 题目背景

此题还有升级版，见[又是整数裂项题目，编号$T604592$](https://www.luogu.com.cn/problem/T604592)

## 题目描述

我们通常把$1×2×3......×a + 2×3......×(a+1)+......+b×（b+1）......×（b+a-1）$这类算式叫做整数裂项.现在小明得到了数$a,b$，但是小明比较懒，想让会编程的你帮他写段代码来解决问题。

## 输入格式

一行，两个整数$a,b$，中间用空格隔开。

## 输出格式

一行，一个整数，表示计算结果

## 输入输出样例 #1

### 输入 #1

```
3 2
```

### 输出 #1

```
30
```

## 输入输出样例 #2

### 输入 #2

```
2 3
```

### 输出 #2

```
20
```

## 说明/提示

### ****样例解释：****
样例$1$：\
$1×2×3+2×3×4=30$\
样例$2$：\
$1×2+2×3+3×4=20$
### ****数据范围：****
对于$100\%$的数据保证$1\le a,b\le 10$
