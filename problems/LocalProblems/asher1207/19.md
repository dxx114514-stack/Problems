# T685286 偶回文数

## 题目描述

我们定义一种数叫偶回文数：\
·这是一个回文数，即这个数从左往右读以及从右往左读都是一样的。\
·这个数的长度是偶数。

现在给定$n$，请你求出$1～n$中所有的偶回文数的数量以及它们的和，答案对$2^{64}$取模。

## 输入格式

一行一个整数$n$。

## 输出格式

一行两个整数，第一个数表示满足条件数的数量，第二个表示它们的和对$2^{64}$取模的结果。中间用空格分开。

## 输入输出样例 #1

### 输入 #1

```
30
```

### 输出 #1

```
2 33
```

## 输入输出样例 #2

### 输入 #2

```
563365
```

### 输出 #2

```
563 154540155
```

## 输入输出样例 #3

### 输入 #3

```
675988723476
```

### 输出 #3

```
675987 223975383563096583
```

## 输入输出样例 #4

### 输入 #4

```
72663468012385
```

### 输出 #4

```
7266346 1240072085532064440
```

## 说明/提示

对于$20\%$的数据保证$1 \le n \le 10^6$。\
对于$40\%$的数据保证$1 \le n \le 5 \times 10^7$。\
对于$100\%$的数据保证$1 \le n \le 10^{14}$。
