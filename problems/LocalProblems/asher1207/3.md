# T604592 又是整数裂项

## 题目描述

我们通常把$1×2×3......×a + 2×3......×(a+1)+......+b×（b+1）......×（b+a-1）$这类算式叫做整数裂项。现在小明得到了数$a,b$，但是小明比较懒，想让会编程的你帮他写段代码来解决问题。

## 输入格式

本题有多组测试数据
第一行一个整数$T$，代表测试数据数量。
针对每组数据，输入一行，两个正整数$a,b$

## 输出格式

针对每组数据，输出一行，表示答案乘$a+1$后对$10^9+7$取模的结果

## 输入输出样例 #1

### 输入 #1

```
3
3 2
100 75
959649 874853
```

### 输出 #1

```
120
656864717
5955211
```

## 说明/提示

对于$7\%$的数据：$1\le a,b\le 10,1\le T\le 5$\
对于$31\%$的数据：$1\le a,b\le 10^3,1\le  T\le 10$\
对于$55\%$的数据：$1\le a,b\le 2 \times10^5,1\le T\le 100$\
对于$100\%$的数据：$1\le a,b\le 10^6,1\le T\le 10^5$
