# T604594 数的因数个数

## 题目描述

给定一个整数$n$，求$n$的因数个数

## 输入格式

一行一个整数$n$

## 输出格式

一行，代表$n$的因数个数

## 输入输出样例 #1

### 输入 #1

```
90
```

### 输出 #1

```
12
```
