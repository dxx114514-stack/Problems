# T604593 mul Problem

## 题目描述

给定两个正整数$a,b$，求$(a×b)$ $mod$ $(10^9+7)$的结果

## 输入格式

一行两个正整数$a,b$

## 输出格式

输出一行一个整数，表示答案

## 输入输出样例 #1

### 输入 #1

```
15 12
```

### 输出 #1

```
180
```
