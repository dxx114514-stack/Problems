# T604596 错题本

## 题目背景

小明在做加减法时总是会有错题，现在给定小明的错题，请你帮忙整理

## 题目描述

小明比较懒，他在输入时用$a$表示加法,用$b$表示减法，你需要按照小明输入的顺序把题目整理出来，一行一道题，格式如下：\
$X+Y=$\
或$X-Y=$\
注：$X$和$Y$是小明读入的数据

## 输入格式

第一行一个整数$n$，表示题目数量
接下来$n$行，每行有两种输入方式：\
```a x y```表示$x+y$\
```b x y```表示$x-y$

## 输出格式

$n$行，每行一个算式，格式见题目描述

## 输入输出样例 #1

### 输入 #1

```
3
a 12 14
a 973 12
b 253 64
```

### 输出 #1

```
12+14=
973+12=
253-64=
```

## 输入输出样例 #2

### 输入 #2

```
4
b 98643 15
a 124 17
a 9 1
b 3 5
```

### 输出 #2

```
98643-15=
124+17=
9+1=
3-5=
```

## 说明/提示

对于全部的数据保证$1\leqslant n \leqslant 100,1\leqslant x,y\leqslant 10^4$
