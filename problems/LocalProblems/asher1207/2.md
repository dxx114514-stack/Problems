# T604591 跳远比赛

## 题目背景

某高中进行跳远比赛，由于人太多了，裁判小$A$想让你对他们的成绩进行排序。

## 题目描述

有$N$个参赛者，其中每个人有$6$次试跳机会（每次试跳都有成绩），按照每个人的最高成绩对他们的成绩进行排序，如果成绩一致，那只能按照人的编号$i$从小到大进行排序（虽然有点不公平）。

## 输入格式

第一行一个整数$N$\
接下来$N$行，每行$6$个浮点数，代表每个人的成绩，第$i$行代表选手编号$i$的成绩。

## 输出格式

$N$行，每行按照以下格式输出
```
No.(i):Student(id),the best junping:(mark)
```
注：括号指填的数

## 输入输出样例 #1

### 输入 #1

```
5
2.81 5.79 2.67 5.10 3.49 5.33 
2.08 2.62 5.30 3.61 5.59 2.04 
3.19 5.64 4.53 2.42 2.72 3.41 
3.75 3.98 3.90 2.80 2.25 2.51 
4.77 4.73 2.67 2.09 5.42 3.80 
```

### 输出 #1

```
No.1:Student1,the best junping:5.79
No.2:Student3,the best junping:5.64
No.3:Student2,the best junping:5.59
No.4:Student5,the best junping:5.42
No.5:Student4,the best junping:3.98
```

## 说明/提示

对于全部的数据保证$：1 \leqslant N\leqslant10^5$
