# T619275 mul Problem plus

## 题目描述

给定$n$和一个长度为$n$的数组$a$，同时给出$L,R$，求出$\prod_{i=L}^R a_i$ $mod$ $100003$的结果。

## 输入格式

第一行，一个整数$n$\
第二行，有$n$个正整数，代表数组$a$\
第三行，一个正整数$Q$，表示询问次数\
对于每组询问，占一行，有两个正整数$L,R$

## 输出格式

对于每组询问，输出一个数表示答案。

## 输入输出样例 #1

### 输入 #1

```
5
4 9 16 3 6
2
1 3
2 5
```

### 输出 #1

```
576
2592
```

## 说明/提示

对于$10\%$的数据：$1 \leq L < R \leq n \leq 10$，$1 \leq a_i \leq 50$，$1 \leq Q \leq 5$\
对于$30\%$的数据：$1 \leq L < R \leq n \leq 10^4$，$1 \leq a_i \leq 10^5$，$1 \leq Q \leq 10^3$\
对于$100\%$的数据：$1 \leq L < R \leq n \leq 5 \times 10^5$，$1 \leq a_i \leq 10^5$，$1 \leq Q \leq 2 \times 10^5$
