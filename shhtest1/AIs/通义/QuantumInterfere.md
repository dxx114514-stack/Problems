# T762057 量子比特的相位干涉

## 题目背景

在量子计算中，相邻量子比特的相位差会影响系统的干涉模式。

## 题目描述

有一个长度为 $n$ 的量子比特序列,每个位置的相位值为 $a_i$（整数）。

定义区间 $[l,r]$ 的**干涉强度**为该区间内所有相邻相位差的绝对值之和，即
$$I(l,r) = \sum_{i=l}^{r-1} |a_{i+1} - a_i|$$

现在进行 $m$ 次操作：

- `1 l r x`：将区间 $[l,r]$ 内所有量子比特的相位值增加 $x$（$x$ 可为负数）。
- `2 l r`：查询区间 $[l,r]$ 的干涉强度 $I(l,r)$。

## 输入格式

第一行两个整数 $n,m$。

第二行 $n$ 个整数 $a_1,a_2,\dots,a_n$。

接下来 $m$ 行，每行一个操作，格式如题所述。

## 输出格式

对于每个操作 `2`，输出一行一个整数，表示干涉强度。

## 输入输出样例 #1

### 输入 #1

```
4 3
1 5 2 4
2 1 4
1 2 3 2
2 1 4
```

### 输出 #1

```
9
9
```

## 说明/提示

- 初始序列 `[1, 5, 2, 4]`。
- 查询 `2 1 4`：$|5-1| + |2-5| + |4-2| = 4 + 3 + 2 = 9$。
- 操作 `1 2 3 2`：区间 $[2,3]$ 增加 2，序列变为 `[1, 7, 4, 4]`。
- 查询 `2 1 4`：$|7-1| + |4-7| + |4-4| = 6 + 3 + 0 = 9$。

注意：区间加操作只影响区间端点处的相邻差值，区间内部的相邻差值保持不变。

### 数据范围

- $2 \le n,m \le 2\times 10^5$
- $|a_i|,|x| \le 10^9$
- 保证 $1 \le l \le r \le n$
