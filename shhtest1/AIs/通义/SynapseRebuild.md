# T762056 神经网络的突触重构

## 题目背景

在人工神经网络中，神经元之间的连接权重会随时间变化。

## 题目描述

有 $n$ 个神经元排成一列，编号 $1\sim n$。初始时，相邻神经元之间有一条突触连接，连接的强度为 $w_i$（表示神经元 $i$ 与 $i+1$ 之间的连接强度）。

定义一段连续神经元区间 $[l,r]$ 的**信号传导效率**为该区间内所有突触强度的最小值，即
$$E(l,r) = \min_{i=l}^{r-1} w_i$$

现在进行 $m$ 次操作：

- `1 l r x`：将区间 $[l,r-1]$ 内所有突触强度增加 $x$（$x$ 可为负数）。
- `2 l r`：查询区间 $[l,r]$ 的信号传导效率 $E(l,r)$。

## 输入格式

第一行两个整数 $n,m$。

第二行 $n-1$ 个整数 $w_1,w_2,\dots,w_{n-1}$。

接下来 $m$ 行，每行一个操作，格式如题所述。

## 输出格式

对于每个操作 `2`，输出一行一个整数，表示信号传导效率。

## 输入输出样例 #1

### 输入 #1

```
5 5
3 1 4 2
2 1 4
1 2 3 2
2 1 4
1 1 2 -1
2 2 5
```

### 输出 #1

```
1
3
2
```

## 说明/提示

- 初始突触强度为 `[3, 1, 4, 2]`。
- 查询 `2 1 4`：区间 $[1,4]$ 对应突触 $w_1,w_2,w_3$，最小值为 $\min(3,1,4)=1$。
- 操作 `1 2 3 2`：$w_2,w_3$ 分别增加 2，变为 `[3, 3, 6, 2]`。
- 查询 `2 1 4`：$\min(3,3,6)=3$。
- 操作 `1 1 2 -1`：$w_1$ 减少 1，变为 `[2, 3, 6, 2]`。
- 查询 `2 2 5`：区间 $[2,5]$ 对应突触 $w_2,w_3,w_4$，$\min(3,6,2)=2$。

### 数据范围

- $2 \le n,m \le 2\times 10^5$
- $|w_i|,|x| \le 10^9$
- 保证 $1 \le l < r \le n$
