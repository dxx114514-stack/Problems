# T764295 间隔约束取权最大值

## 题目背景

豆包

## 题目描述

给定长度为n的权值序列，从中挑选若干元素。
挑选约束：任意两个被选中元素的下标差值不能小于给定距离d。
计算可以选出的元素总和的最大数值。

## 输入格式

第一行两个整数 n,d
第二行n个整数，代表序列每个位置的权值

## 输出格式

输出一个整数，表示选取元素能得到的最大权值总和

## 输入输出样例 #1

### 输入 #1

```
6 2
5 1 3 4 2 6
```

### 输出 #1

```
11
```

## 说明/提示

$1\le n \le3000,1\le d\le n,-10^4\le val\le 10^4$
