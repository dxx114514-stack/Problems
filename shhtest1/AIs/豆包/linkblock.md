# T764297 代价限制连通统计

## 题目背景

豆包

## 题目描述

共有n个节点，给出m条带权无向边。
仅允许合并边权小于等于阈值lim的两个节点，边权超阈值无法连通。
计算完成合法合并后，整张图内互不连通的独立块总数量。

## 输入格式

第一行三个整数 n,m,lim
接下来m行，每行三个整数u,v,w，代表u、v节点间存在权值为w的无向边

## 输出格式

输出一个整数，表示最终独立连通块的个数

## 输入输出样例 #1

### 输入 #1

```
5 4 6
1 2 3
2 3 7
3 4 2
4 5 8
```

### 输出 #1

```
3
```

## 说明/提示

$1\le n,m\le10^5,1\le 权值,lim\le 10^9$
