# T764291 限长区间 K 和计数

## 题目背景

豆包

## 题目描述

给定包含正负整数的序列、目标值K、区间最大长度限制L。
统计同时满足以下两个条件的连续子区间总个数：
1. 子区间所有元素相加和恰好等于K
2. 子区间包含元素数量不超过L

## 输入格式

第一行输入三个整数 n,K,L
第二行输入n个整数，代表给定整数序列

## 输出格式

输出一个整数，表示合法子区间的总数量

## 输入输出样例 #1

### 输入 #1

```
5 3 3
1 2 -1 3 0
```

### 输出 #1

```
4
```

## 说明/提示

### 数据范围
$1\le n\le 2000,-10^5\le a_i,K\le10^5,1\le L\le n$
