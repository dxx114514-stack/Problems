# T764293 分组载重临界判定

## 题目背景

豆包

## 题目描述

现有n件货物，每件拥有对应重量。需要将货物划分成连续的组，最多划分m个组。
要求每一组内所有货物总重量不超过统一载重上限。
求出满足分组条件的最小载重上限数值。

## 输入格式

第一行两个整数 n,m
第二行n个整数，依次为每件货物的重量

## 输出格式

输出一个整数，代表符合要求的最小载重值

## 输入输出样例 #1

### 输入 #1

```
6 3
7 2 5 1 3 4
```

### 输出 #1

```
11
```

## 说明/提示

$1\le n\le 10^5,1\le m\le n,1\le w_i\le10^6$
