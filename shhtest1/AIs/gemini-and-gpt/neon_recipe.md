# T762179 霓虹配方

## 题目背景

霓虹城的药剂师把配方写在一排发光路标上。每个路标都能把当前半成品送往后面若干个路标，但光效会随距离产生平方级损耗。更麻烦的是，每个路标的配方只在一段后继位置内有效。

你要调出最后一盏霓虹灯的颜色，并让总损耗尽量小。

## 题目描述

有 $n$ 个配方站点，按坐标从小到大排列。第 $i$ 个站点坐标为 $x_i$，满足

$$
x_1 < x_2 < \cdots < x_n.
$$

你从站点 $1$ 开始，初始代价为 $0$。如果当前在站点 $i$，可以跳到任意站点 $j$，满足

$$
i < j \le r_i.
$$

这次跳转的代价为

$$
b_i + (x_j-x_i)^2 + c_j.
$$

其中 $b_i$ 是使用站点 $i$ 的配方代价，$c_j$ 是进入站点 $j$ 的稳定代价。站点 $1$ 的 $c_1$ 不计入初始代价。

请输出从站点 $1$ 到站点 $n$ 的最小总代价。如果无法到达站点 $n$，输出 `-1`。

## 输入格式

第一行一个整数 $n$。

接下来 $n$ 行，每行四个整数 $x_i,c_i,b_i,r_i$。

## 输出格式

输出一个整数，表示最小总代价；若无法到达，输出 `-1`。

## 输入输出样例 #1

### 输入 #1

```
5
1 0 3 3
3 4 2 5
6 1 5 5
10 3 1 5
11 2 0 5
```

### 输出 #1

```
51
```

## 说明/提示

### 样例解释

一种最优路径是 $1\to 2\to 3\to 4\to 5$：

- $1\to 2$：代价 $3+(3-1)^2+4=11$；
- $2\to 3$：代价 $2+(6-3)^2+1=12$；
- $3\to 4$：代价 $5+(10-6)^2+3=24$；
- $4\to 5$：代价 $1+(11-10)^2+2=4$。

总代价为 $51$。

### 数据范围

对于所有测试点：

- $2 \le n \le 5\times 10^4$
- $0 \le c_i,b_i \le 10^9$
- $1 \le x_1 < x_2 < \cdots < x_n \le 10^6$
- $i \le r_i \le n$
