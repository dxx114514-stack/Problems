## 题解：量子纠缠的纠缠熵

### 核心观察

相邻关系只存在于 $i$ 与 $i+1$ 之间。若 $a_i \oplus a_{i+1} &gt; k$，则 $i$ 与 $i+1$ 之间有一条**断裂边**。

断裂边将序列划分为若干个连续的纠缠簇。设各簇长度为 $L_1, L_2, \dots, L_t$，则
$$S = \sum L_i^2$$

修改位置 $p$ 的自旋值，只会影响两条相邻边：$(p-1,p)$ 与 $(p,p+1)$。因此最多只有两个断裂点会发生变化，最多导致三个相邻簇合并或分裂。

### 算法

用 `std::set` 维护所有断裂边的位置（存储左端点下标 $i$，表示 $i$ 与 $i+1$ 之间断裂）。同时维护当前的总纠缠熵 $S$。

- 初始 $O(n)$ 扫描，建立断裂边集合，并计算初始 $S$。加入哨兵断裂边 $-1$ 和 $n-1$（表示首尾外侧）。
- 修改位置 $p$：
  1. 找到 $p$ 所在簇（通过 set 的 `upper_bound` 找到左右断裂边）。
  2. 将该簇的长度贡献从 $S$ 中减去。
  3. 检查并更新边 $(p-1,p)$ 与 $(p,p+1)$ 的断裂状态（注意边界）。
  4. 重新计算受影响后的簇长度，将新贡献加回 $S$。
- 查询操作直接输出 $S$。

### 复杂度

- 时间：$O((n+m)\log n)$，set 操作与常数次簇长度更新。
- 空间：$O(n)$。
