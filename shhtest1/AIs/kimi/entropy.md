# T762050 量子纠缠的纠缠熵

## 题目描述

在一维量子自旋链上有 $n$ 个粒子，第 $i$ 个粒子的自旋值为 $a_i$（非负整数）。

若两个相邻粒子 $i$ 与 $i+1$ 满足 $a_i \oplus a_{i+1} \le k$（$\oplus$ 为按位异或），则称它们处于**纠缠态**。由纠缠态相邻关系连接的极大连续区间称为一个**纠缠簇**。

物理学家定义整个系统的**纠缠熵**为所有纠缠簇大小的平方和，即
$$S = \sum_{\text{簇 }C} |C|^2$$

现在进行 $m$ 次操作：

- `1 i x`：将第 $i$ 个粒子的自旋修改为 $x$。
- `2`：查询当前系统的纠缠熵 $S$。

## 输入格式

第一行三个整数 $n,m,k$。

第二行 $n$ 个整数 $a_1,a_2,\dots,a_n$。

接下来 $m$ 行，每行一个操作，格式如题所述。

## 输出格式

对于每个操作 `2`，输出一行一个整数，表示当前的纠缠熵 $S$。

## 输入输出样例 #1

### 输入 #1

```
5 5 3
1 2 3 0 1
2
1 3 7
2
1 4 2
2
```

### 输出 #1

```
25
9
9
```

## 说明/提示

## 数据范围

- $1 \le n,m \le 2\times 10^5$
- $0 \le k, a_i, x < 2^{30}$
