## 题解：密码学的模幂运算链（ModPowChain）

### 核心观察

计算幂塔 $a_l^{a_{l+1}^{\cdots^{a_r}}} \bmod P$ 需要使用**欧拉降幂**。

根据欧拉定理：若 $\gcd(a, P) = 1$，则 $a^{\phi(P)} \equiv 1 \pmod P$。

对于一般的幂运算 $a^b \bmod P$：
- 如果 $b < \phi(P)$，直接计算 $a^b \bmod P$。
- 如果 $b \ge \phi(P)$，则 $a^b \equiv a^{b \bmod \phi(P) + \phi(P)} \pmod P$。

由于幂塔增长极快，从右到左计算时，指数很快就会超过 $\phi(P)$。

### 算法

使用**递归版欧拉降幂**从右到左计算幂塔。

**核心函数** `calc_power_tower(a, l, r, mod)`：
1. 递归基：如果 $l = r$，返回 $a_l \bmod mod$
2. 如果 $mod = 1$，返回 0
3. 计算 $\phi(mod)$
4. 递归计算指数部分：$exp = calc\_power\_tower(a, l+1, r, \phi(mod))$
5. 判断指数是否需要 $+ \phi(mod)$：
   - 如果幂塔高度 $\ge 5$，指数肯定很大，直接 $exp = exp \bmod p + p$
   - 否则需要实际判断是否溢出
6. 返回 $a_l^{exp} \bmod mod$

**关键实现细节**：
- 使用 `__int128` 防止快速幂中间结果溢出
- 对于小幂塔（高度 < 5），需要精确判断指数是否 $\ge \phi(mod)$
- 对于大幂塔（高度 $\ge 5$），可以直接认为指数很大

对于单点修改，直接更新数组即可。

### 复杂度

- 时间：每次查询 $O((r-l) \log P)$，需要递归计算 $\phi$ 和快速幂。
- 空间：$O(r-l)$ 递归栈深度。

### 注意事项

- 需要处理 $\gcd(a, P) \neq 1$ 的情况（通过 $+ \phi(mod)$ 解决）。
- 幂塔高度较大时指数必然超过 $\phi(mod)$，可以直接应用欧拉降幂。
- 使用 `__int128` 类型进行快速幂运算，防止 $10^{18} \times 10^{18}$ 溢出。
