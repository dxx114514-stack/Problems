# T762059 密码学的模幂运算链

## 题目背景

在公钥密码系统中，模幂运算是核心操作之一。

## 题目描述

有一个长度为 $n$ 的整数序列 $a_1, a_2, \dots, a_n$，以及一个固定的模数 $P$。

定义区间 $[l,r]$ 的**模幂链值**为：
$$C(l,r) = a_l^{a_{l+1}^{a_{l+2}^{\cdots^{a_r}}}} \bmod P$$

即从右到左依次进行幂运算，最后对 $P$ 取模。

现在进行 $m$ 次操作：

- `1 i x`：将 $a_i$ 修改为 $x$。
- `2 l r`：查询区间 $[l,r]$ 的模幂链值 $C(l,r)$。

## 输入格式

第一行三个整数 $n,m,P$。

第二行 $n$ 个整数 $a_1,a_2,\dots,a_n$。

接下来 $m$ 行，每行一个操作，格式如题所述。

## 输出格式

对于每个操作 `2`，输出一行一个整数，表示模幂链值。

## 输入输出样例 #1

### 输入 #1

```
4 3 1000000007
2 3 2 2
2 1 4
1 3 4
2 1 4
```

### 输出 #1

```
993282280
688434784
```

## 说明/提示

- 初始序列 `[2, 3, 2, 2]`，$P = 10^9+7$。
- 查询 `2 1 4`：计算 $2^{3^{2^2}} \bmod P = 2^{3^4} \bmod P = 2^{81} \bmod P$。
  - $2^{81} = 2417851639229258349412352$，对 $10^9+7$ 取模后为 $993282280$。
  
- 操作 `1 3 4`：序列变为 `[2, 3, 4, 2]`。
- 查询 `2 1 4`：计算 $2^{3^{4^2}} \bmod P = 2^{3^{16}} \bmod P = 2^{43046721} \bmod P$。
  - 对 $10^9+7$ 取模后为 $688434784$。

### 数据范围

- $1 \le n,m \le 2\times 10^5$
- $2 \le P \le 10^9+7$，$P$ 是质数
- $1 \le a_i, x \le 10^9$
- 保证 $1 \le l \le r \le n$
- **注意**：由于幂塔增长极快，实际计算时需要使用欧拉降幂等技巧
