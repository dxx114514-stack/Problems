# 潮汐门

## 题目背景

海面下有一座被潮汐驱动的旧城。城中的门并不会一直开启，而是按照同一个周期反复张合。  
守门人给了你每扇门的潮汐表：只要出发时刻落在允许的余数区间里，就可以穿过它。

## 题目描述

有一个 $n$ 个点、$m$ 条有向边的图，所有时间均为非负整数。第 $i$ 条边从 $u_i$ 到 $v_i$，通过耗时 $w_i$。整座城市有公共周期 $P$。

边 $i$ 只允许在出发时刻 $t$ 满足

$$
l_i \le t \bmod P \le r_i
$$

时进入。你可以在任意点等待任意非负整数时间。

你在 $0$ 时刻位于 $1$ 号点，问最早能在什么时刻到达 $n$ 号点。如果无法到达，输出 `-1`。

图中可以有重边。

## 输入格式

第一行三个整数 $n,m,P$。

接下来 $m$ 行，每行五个整数 $u_i,v_i,w_i,l_i,r_i$，表示一条有向边。

保证 $0 \le l_i \le r_i < P$。

## 输出格式

输出一个整数，表示最早到达 $n$ 号点的时刻；若无法到达，输出 `-1`。

## 样例

### 样例输入

```text
4 5 6
1 2 3 0 2
2 4 4 3 5
1 3 2 1 1
3 4 3 3 3
2 3 1 0 5
```

### 样例输出

```text
6
```

## 样例解释

一种最优方案是：

- 在点 $1$ 等到时刻 $1$，此时 $1 \bmod 6=1$，可以经过边 $1\to 3$，到达点 $3$ 的时刻为 $3$；
- 时刻 $3$ 满足 $3 \bmod 6=3$，可以经过边 $3\to 4$，到达点 $4$ 的时刻为 $6$。

不存在更早到达点 $4$ 的方案。

## 数据范围

对于所有测试点：

- $2 \le n \le 10^5$
- $1 \le m \le 2\times 10^5$
- $1 \le P \le 60$
- $1 \le w_i \le 10^9$
- $1 \le u_i,v_i \le n,\ u_i\ne v_i$

## 计分方式

本题共 20 个测试点，每个测试点 5 分。

## 时间与空间限制

- 时间限制：2 秒
- 空间限制：256 MB
